Frequently Asked Questions

Do I need a Riot API key?

No. Radianite is designed around your local Riot Client and active VALORANT session.

Does this ask for my Riot password?

No. Radianite does not need your Riot password. It reads local session information from the Riot Client already running on your PC.

Why does the app say Riot Client is closed?

Start Riot Client first. If VALORANT is not running yet, Radianite may show a waiting or partial status until the game session becomes available.

How do I add the overlay to OBS?

Copy the overlay URL from Radianite, open OBS, add a Browser Source, and paste the URL. Keep Radianite running while streaming.

Does Discord Rich Presence require Discord to be open?

Yes. Discord Rich Presence works through the Discord desktop app, so Discord must be running.

Is this allowed by Riot?

Radianite is intended to be a local companion and overlay tool. You are responsible for following Riot's terms and any tournament or platform rules that apply to you.
